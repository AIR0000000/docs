## Setu eStamp

### What is eStamp?

Our eStamp solution enable easy aquisition of offline eStamps.  
Attach it to a document and perform aadhaar based electronic signature on top of it.

All features of the setu esign is available in eStamp flow as well.
<hr class="primary" />

### What can I do with this product?
Setu Aadhaar-Based eSign gateway supports adding government stamp duty on the fly. 
Our eSign APIs empower you to collect legally bound aadhaar based eSign while adding government 
stamp duties ensuring compliance and security on documents from signers. With these APIs, you can streamline your 
customer onboarding process and close contracts digitally, all while providing a frictionless experience

## Setu eStamp integration

To use eStamp solution

1. **Raise eStamp aquire request to Setu team (support@setu.co)** —Raise request for count, value and Indian state of the stamp duty needed.
2. **Create signature request and complete signing**— Use the esigning flow to complete signing and download signed document.


### Flow of API calls
Creating a eStamp request is same as creating a signature request.
The flow of api calls is mentioned <a href="/data/esign/quickstart">here.</a>


<br/>

#### Create Signature request with eStamp


Signature request can be creared with defined eStamp value 
and indian state. This eStamp will be attached to the document that you upload for signing.
<br/>

Here is a quick description of the additional values required, to be passed for signing a document with eStamp.

- `eStampState` the Indian state of the eStamp.
- `eStampValue` the value of the eStamp 
- `eStampMergePosition` supported values are -1 and 0. -1 indicates that eStamp pdf would be merged at end of document, 0 indicates the eStamp pdf will be merged at the start of the document.
