## Estamp

### What is Estamp?

Our Estamp solution enable easy aquisition of offline Estamps.  
Attach it to a document and perform aadhaar based electronic signature on top of it.

All features of the setu esign is available in Estamp flow as well.
<hr class="primary" />

### What can I do with this product?

Offer your customers a seamless experience of Aadhaar based electronic signatures on goverment issuesd stmaps.

Whichever industry you belong to—banking, insurance, finance, law and more—you can provide 
smooth onboarding experiences for your customers and enable contracts to be closed digitally.

## Setu Estamp integration

To use estamp solution

1. **Raise estamp aquire request to Setu integeration team**—Raise request for count of stamps you need.
2. **Create signature request and complete signing**— Use the esigning flow to complete signing and download signed document.


### Flow of API calls

Read more about the flow of API calls in esign <a href="/data/esign/quickstart">here</a>


<br/>

#### Create Signature request with estmap


Signature request can be creared with defined estamp value 
and state. This stamp will be attached to the document that you upload for singing.
<br/>

Here is a quick description of the additionl values required to use for estamp while signing—

- `estampState` the state of the stamp.
- `estampValue` the value of the stamp 
- `estampMergePosition` supported values are -1 and 0 -1 indicates at the end, 0 is to merge the stamp at start of document
