## Mandate Operations
A UPI Mandate itself is not a transaction, but rather a standing instruction on which transactions are carried out. Therefore, a status of a mandate can change just like
updating an agreement between two parties. 

There are four main operations that can happen on a UPI mandate after creation:
<ul>
    <li>Updating a mandate. Read more <a href="/payments/umap/mandates/generic/update" target="_blank">here</a></li>
    <li>Revoking a mandate. Read more <a href="/payments/umap/mandates/generic/revoke" target="_blank">here</a></li>
    <li>Pausing a mandate. Read more <a href="/payments/umap/mandates/generic/pause" target="_blank">here</a></li>
    <li>Unpausing a paused mandate. Read more <a href="/payments/umap/mandates/generic/unpause" target="_blank">here</a></li>
</ul>
