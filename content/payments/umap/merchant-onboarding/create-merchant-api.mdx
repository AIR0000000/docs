## Create merchant

You can use the following API, to on-board merchants in UMAP, by providing required business details for merchants and your `aggregatorID`.

<Callout type='tip'>For merchants acquired directly, <code>aggregatorID</code> can be accessed from Bridge.</Callout>
<br/>

<table>
    <tbody>
        <tr>
            <th>Staging host</th>
            <td>
                <code>https://umap-dev-core.setu.co/api</code>
            </td>
        </tr>
        <tr>
            <th>Method</th>
            <td>
                <code>POST</code>
            </td>
        </tr>
        <tr>
            <th>Path</th>
            <td>
                <code>/v1/aggregators/merchants</code>
            </td>
        </tr>
        <tr>
            <th>Header</th>
            <td>

                Generate this token using <a

                    target="_blank">
                    OAuth
                </a> or <a
                    href="/payments/upi-deeplinks/resources/jwt"

                </a>.<br />

    "businessName": "deku",

    "legalName": "deku",
    "mcc": "5432",
