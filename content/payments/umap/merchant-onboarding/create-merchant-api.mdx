## Create Merchant

`TODO: change to wrapper API`

Create merchants in UPI with this API by providing business details & aggregator ID for onboarding and enabling transactions for the merchant.

<table>
    <tbody>
        <tr>
            <th>Method</th>
            <td>
                <code>POST</code>
            </td>
        </tr>
        <tr>
            <th>Path</th>
            <td>
                <code>aggregators/wrapper/merchants</code>
            </td>
        </tr>
        <tr>
            <th>Header</th>
            <td>
                <code>Authorization</code>: Bearer &lt;<b>insert_token_here</b>&gt;.
                Generate this token using <a
                    href="/payments/upi-deeplinks/resources/oauth"
                    target="_blank">
                    OAuth
                </a> or <a
                    href="/payments/upi-deeplinks/resources/jwt"
                    target="_blank">
                    JWT
                </a>.<br />
                <br />
                <code>Content-Type</code>—will be <code>application/json</code>
            </td>
        </tr>
    </tbody>
</table>
