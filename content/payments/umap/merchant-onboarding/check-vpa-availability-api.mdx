## Check availability of VPA 
VPAs act as unique identifiers for merchants in the UPI ecosystem, so it is highly recommended that you check VPA validity before assigning it to a merchant, to avoid conflicts. 

Use the following API to check if it is already associated with another merchant or not.

<table>
    <tbody>
        <tr>

#### Sample request

<CodeBlockWithCopy language="json">
    {`{
    "vpa": "deku@okokaxis"
}`}
</CodeBlockWithCopy>

  "status": "available"
}`}
</CodeBlockWithCopy>
