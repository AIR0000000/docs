## Create Payment Link

<p>The merchant will have to call the <b><i>create payment link API</i></b> with the correct set of parameters.</p>

<table>
    <tbody>
        <tr>
            <th>Staging Host</th>
            <td>
                <code>https://umap-dev-core.setu.co/api</code>
            </td>
        </tr>
        <tr>
            <th>Method</th>
            <td>
                <code>POST</code>
            </td>
        </tr>
        <tr>
            <th>Path</th>
            <td>
                <code>/v1/merchants/payment-links</code>
            </td>
        </tr>
        <tr>
            <th>Header</th>
            <td>
                <code>merchantId</code><br/>
                <code>Authorization</code>: Bearer &lt;<b>insert_token_here</b>&gt;.
                Generate this token using <a
                    href="/payments/upi-deeplinks/resources/oauth"
                    target="_blank">
                    OAuth
                </a> or <a
                    href="/payments/upi-deeplinks/resources/jwt"
                    target="_blank">
                    JWT
                </a>.<br />
                <br />
                <code>Content-Type</code>—will be <code>application/json</code>
            </td>
        </tr>
        <tr>
            <th>API Reference</th>
            <td>
                <a href="/payments/umap/api-reference#/category~payment-link/operation~payment-link#create"target="_blank">
                    Create Payment Link ↗
                </a>
            </td>
        </tr>
    </tbody>
</table>

###### Sample request

<Tabs
  tabs={[
    {
      key: "1",
      label: "Pay Single",
      content: (
        <>
          <CodeBlockWithCopy language="json">
            {`{
  "amount": 10000,
  "currency": "INR",
  "expiryDate": "2023-09-04 12:08:20.202644 +0530 IST",
  "merchantReferenceId": "1240689979139556557",
  "merchantVpa": "deku@okokaxis",
  "metadata": {
      "InvoiceNumber": "INV-21-12-009"
  },
  "paymentType": "pay_single",
  "productConfigId": "01ARZ3NDEKTSV4RRFFQ69G5FAV",
  "transactionNote": "testpay"
}`}
          </CodeBlockWithCopy>
          <Callout type="note">
            <strong>amount</strong> is a mandatory field
          </Callout>
          <br/>
          <p>By creating a payment link with paymentType as <code>pay_single</code>
            you can accept the payment only once.
          </p>
        </>
      ),
    },
     {
      key: "2",
      label: "Pay Multi",
      content: (
        <>
          <CodeBlockWithCopy language="json">
            {`{
  "currency": "INR",
  "expiryDate": "2023-09-04 12:08:20.202644 +0530 IST",
  "merchantReferenceId": "1240689979139556557",
  "merchantVpa": "deku@okokaxis",
  "metadata": {
      "InvoiceNumber": "INV-21-12-009"
  },
  "paymentType": "pay_multi",
  "productConfigId": "01ARZ3NDEKTSV4RRFFQ69G5FAV",
  "transactionNote": "testpay"
}`}
          </CodeBlockWithCopy>
         <Callout type="note">
            <strong>amount</strong> is not required when creating multi use payment link
          </Callout>
          <br/>
          <p>By creating a payment link with paymentType as <code>pay_multi</code>
            you can accept the payment more than once.
          </p>
        </>
      ),
    }
  ]}
/>

## Check Payment Link Status

<p>The merchant can also get the payment link status by payment link id by calling <b><i>check payment link status API</i></b></p>

<table>
    <tbody>
         <tr>
            <th>Staging Host</th>
            <td>
                <code>https://umap-dev-core.setu.co/api</code>
            </td>
        </tr>
        <tr>
            <th>Method</th>
            <td>
                <code>GET</code>
            </td>
        </tr>
        <tr>
            <th>Path</th>
            <td>
                <code>/v1/merchants/payment-links/&#123;id&#125;</code>
            </td>
        </tr>
        <tr>
            <th>Header</th>
            <td>
                <code>merchantId</code><br/>
                <code>Authorization</code>: Bearer &lt;<b>insert_token_here</b>&gt;.
                Generate this token using <a
                    href="/payments/upi-deeplinks/resources/oauth"
                    target="_blank">
                    OAuth
                </a> or <a
                    href="/payments/upi-deeplinks/resources/jwt"
                    target="_blank">
                    JWT
                </a>.<br />
                <br />
                <code>Content-Type</code>—will be <code>application/json</code>
            </td>
        </tr>
        <tr>
            <th>API Reference</th>
            <td>
                <a href="/payments/umap/api-reference#/category~payment-link/operation~payment-link#fetchById"target="_blank">
                    Check Payment Link Status ↗
                </a>
            </td>
        </tr>
    </tbody>
</table>
