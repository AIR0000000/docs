## FAQs and Troubleshooting

<hr class="primary" />

##### How do I configure Merchant Payment Return URL not configured in Dashboard?

Configure from Setting->General->Payment Return URL

<hr class="primary" />

##### How do I configure Payment Gateway on Console?

On Merchant Dashboard. Smart Routing --> Create Gateway

<hr class="primary" />

##### When is No Acquirer Found error thrown?

When no Payment Gateway is configured. Please configure a payment gateway by going to Smart Routing --> Create Gateway

<hr class="primary" />

##### How to add multiple merchant url on merchant dashboard?

Configure from Setting->General->Payment Return URL separated by comma(,)without any spaces.

<hr class="primary" />

##### How to add one or multiple Gateways in Dashboard?

Create Gateways-> Select gateway from dropdown->Configure gateway credentials->create

<hr class="primary" />

##### How to set Routing Scores for bin, issuer, card brand?

Create route->enter required details->create

<hr class="primary" />

##### How to set routing preference for gateways?

Gateway->ROUTES SCORES->Please specify the routing preference of gateways in a comma-separated manner

<hr class="primary" />

##### How to set custom priority logic?

Gateway->PRIORITY LOGICS->enable tom priority logic option

<hr class="primary" />

##### How to create user and role?

User->create user->fill required details->click create user button.

<hr class="primary" />

##### How to get plural credentials for transaction?

Ans : Go to Settings --> Credentials. You will find your MID, Secret Key and Access code

<hr class="primary" />

##### How to set webhook url on dashboard?

Go to Settings --> Webhooks --> Enter url --> Enable webhook events --> Update
